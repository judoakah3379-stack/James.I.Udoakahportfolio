# James Udoakah — Portfolio

A multi-page personal portfolio site for James Udoakah (Software Engineer, Gameplay Engineer, Sound Architect). Built with plain HTML, CSS, and JavaScript, bundled via Vite.

## Pages

- `index.html` — Home / hero
- `about.html` — About
- `projects.html` — Projects
- `planner.html` — Planner
- `contact.html` — Contact

## Asset structure

- `css/style.css` — all styles
- `js/nav.js` — navigation toggle
- `js/animations.js` — scroll / reveal animations
- `js/contact.js` — contact form logic
- `js/planner.js` — planner logic
- `favicon.svg` — site icon

## How to run

```
PORT=5000 BASE_PATH=/ pnpm run dev
```

The workflow **Start application** handles this automatically. Vite serves the site on port 5000.

## Stack

- Vite (bundler / dev server)
- Plain HTML / CSS / JS (no framework)
- pnpm (package manager)
